# Product Overview

## Description

Node.js service integrating with Corellium virtual device platform via MQTT messaging and the Corellium CoreModel API.

## Purpose

Provide real-time communication and control of Corellium virtual devices through message-driven architecture.

## Key Features

- MQTT-based messaging for device events and commands
- Integration with Corellium CoreModel for device management
- Message queue processing for reliable async operations
- Real-time device state synchronization

## Target Users

- Developers building automation on Corellium platform
- DevOps teams managing virtual device fleets
- QA engineers running automated device tests
